# Программный комплекс распознавания голосовых команд на базе Wav2Vec2

Данный репозиторий содержит программный комплекс распределенной системы голосового управления, использующий оптимизированную модель распознавания речи Wav2Vec2 для классификации команд.

## 📁 Структура проекта

*   `Model.py` — инференс модели Wav2Vec2, получение эмбеддингов (768-мерных векторов) и их классификация с помощью MLP.
*   `Optimization.py` — оптимизация модели: экспорт в формат ONNX и квантование весов до INT8 для ускорения инференса.
*   `Srav.py` — скрипт сравнительного анализа точности и времени работы различных версий моделей.
*   `bot_vk.py` — управляющий чат-бот ВКонтакте на базе конечного автомата (State Machine), принимающий голосовые сообщения.
*   `deploy_vds.py` — серверное приложение (API) для развертывания на VDS с базой данных SQLite для хранения команд.
*   `raspberry_client.py` — клиентское ПО для аппаратной платформы Raspberry Pi для выполнения физических команд (светодиоды, звук).
*   `config.json` — конфигурация параметров подключения и токенов.
*   `command_db.json` — локальный реестр зарегистрированных голосовых команд.

---

## ⚙️ Архитектура и оптимизация модели
Инференс построен на модели **Wav2Vec2** (семейство моделей распознавания речи). Для возможности выполнения в реальном времени на CPU маломощных устройств:
1.  Модель экспортирована в **ONNX**.
2.  Применено квантование весов (**INT8 Quantization**), что снизило размер с 208 МБ до 79 МБ и увеличило скорость работы.
3.  Поверх 768-мерного вектора эмбеддинга обучается легкий полносвязный классификатор (MLP 128x64) для распознавания конкретных команд пользователя.

## 🛠️ Установка и запуск

### Требования
*   Python 3.9+
*   FFmpeg (для обработки голосовых сообщений)

### Установка зависимостей
```bash
pip install -r requirements.txt
```
*(Необходимые библиотеки: `onnxruntime`, `librosa`, `vk_api`, `requests`, `numpy`)*

### Запуск модулей
*   **Запуск сервера VDS:**
    ```bash
    python deploy_vds.py
    ```
*   **Запуск бота ВКонтакте:**
    ```bash
    python bot_vk.py
    ```
*   **Запуск Raspberry Pi клиента:**
    ```bash
    python raspberry_client.py
    ```
